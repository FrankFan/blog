## Getting started

1. Clone the repository: `git clone https://github.com/FrankFan/blog.git`
2. Install dependencies: `yarn`
3. Rename .env.example to .env.local and Modify its content, [get your token](https://github.com/settings/tokens)
4. Start dev server: `yarn dev`
