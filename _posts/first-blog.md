# 搭建 Mac 环境记录

## 一、安装软件

1. 安装 node.js 环境

    使用 [nvm](https://github.com/nvm-sh/nvm) 管理 node 版本

    使用 [nrm](https://github.com/Pana/nrm) 管理 npm 版本

2. 安装 iTerm 环境

     依赖  brew , 安装 HomeBrew，基于oh-my-zsh进行配置：

    安装插件

        1）zsh-autosuggestions

        2）zsh-syntax-highlighting

    参考文章：[https://sspai.com/post/63241](https://sspai.com/post/63241)

## 二、注意事项

打开任意来源选项：  `sudo spctl --master-disable`

安装 Command Line Tools 工具：`xcode-select --install`

xcode 安装参考： [https://cloud.tencent.com/developer/article/1332172](https://cloud.tencent.com/developer/article/1332172)

## 三、快捷键

### 锁屏

1. `CTRL + SHIFT + Power`(或`Eject` 键盘右上角键）
2. `CTRL + CMD + Q` 立即锁定屏幕
